## Planned activity patterns

### Content consumption and exploration

- Watch a screencast
- Review slides
- Reveal a hint
- Browse a glossary
- Review intended learning outcomes

### Assessment and reflection

- Take a poll
- Answer an open question
- Write an essay
- Evaluate yourself
- Check yourself
- Complete an expert-graded assessment
- Provide feedback
- Review a peer’s work

### Social and synchronous learning

- Attend a live class
- Schedule a live session
- Work in breakout rooms
- Contribute to a discussion
- Contribute to a word cloud
- Contribute to a collaborative whiteboard
- Play a dialogue
- Complete a bridge-in activity

### Interactive media and practice

- Interact with video
- Interact with an image
- Compare two images
- Practise in a virtual lab

## Page structure

Each activity is represented by one `.mdoc` page. Its body may include any combination of these checklist subsections:

- Learning content design
- Interaction and interface design
- System design

The page may also include `Related` and `Sources` sections. See [`docs/CONTENT_MODEL.md`](../docs/CONTENT_MODEL.md).
