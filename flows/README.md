# Flows

Flows describe how learning activities are sequenced, paced, released, revisited, or coordinated across learners and facilitators.

## Planned learning flows

### Instructional models

- BOPPPS model
- CARD model
- Repetitive learning and the memory curve
- Microlearning

### Sequence and navigation

- Linear flow
- Non-linear flow
- Branching
- Drip content

### Pace and participation

- Self-paced learning
- Cohort-based learning
- Synchronous learning
- Asynchronous learning

### Course lifecycle

- Starting a course
- Returning to a course

### Assessment flow

- Quiz feedback
- Attempt limits
- Time limits
